# CSS Art: Heart

A delightful pure CSS animation featuring a cute cartoon animal holding a heart. This project showcases the creative possibilities of CSS for creating animated illustrations without any JavaScript or images.

## Preview

![Screenshot](Screenshot%202026-04-13%20022924.png)

## How to View

Simply open `index.html` in your web browser to see the animated heart art in action.

## Features

- Pure CSS animation
- No JavaScript required
- Accessible with ARIA labels
- Lightweight and fast-loading

## Technologies Used

- HTML5
- CSS3 (animations and transforms)

Enjoy the cute animation!
